## 1.2 越界风险识别与应对

### 1.2.1 潜在越界风险清单

**高风险（需立即处理）**：

1. **策略漂移风险**
   - **风险描述**：实盘运行中策略参数可能因市场变化而逐渐偏离最优区间
   - **越界表现**：胜率持续下降、回撤扩大但未触发熔断
   - **当前状态**：⚠️ 未建立自动检测机制
   - **应对措施**：建立周度策略健康度检查，胜率<50%持续2周自动触发回测重评

2. **Token预算超支风险**
   - **风险描述**：高优先级任务（如紧急风控）可能挤占日常预算
   - **越界表现**：非交易时段Token不足，无法执行盘后分析
   - **当前状态**：⚠️ 动态配额机制未细化
   - **应对措施**：建立"战略储备"机制，保留500万Token作为应急预算

3. **数据覆盖盲区风险**
   - **风险描述**：新标的回测数据不足，导致策略适配度评估失真
   - **越界表现**：影子模式验证通过，但实盘表现差异大
   - **当前状态**：⚠️ 数据湖建设刚启动
   - **应对措施**：新标的必须满足≥6个月历史数据方可进入备选池

**中风险（需规划处理）**：

4. **多策略冲突风险**
   - **风险描述**：BTDR和连连数字同时触发交易信号，可能争夺资金
   - **越界表现**：资金分配不均，单策略过度集中
   - **当前状态**：✅ 已通过独立风控隔离
   - **优化方向**：建立资金池动态分配算法

5. **记忆膨胀风险**
   - **风险描述**：长期运行后MEMORY.md文件过大，影响加载效率
   - **越界表现**：会话启动时间延长，上下文压缩频繁
   - **当前状态**：⚠️ 未建立归档机制
   - **应对措施**：季度MEMORY.md归档，超过5000行触发分卷

**低风险（持续监控）**：

6. **外部依赖失效风险**
   - 富途API服务中断
   - GitHub访问受限
   - 企业微信通知失败

### 1.2.3 低效逻辑诊断

**已识别的低效逻辑**：

1. **策略信号生成效率**
   - **现状**：每次信号判断需查询实时行情+计算多指标
   - **低效点**：重复计算相同指标（如20日均线）
   - **优化**：建立指标缓存，均线等慢变指标每5分钟更新一次

2. **回测数据加载效率**
   - **现状**：每次回测从API获取数据
   - **低效点**：网络延迟+API限流
   - **优化**：数据湖本地存储，优先读取本地CSV/SQLite

3. **记忆检索效率**
   - **现状**：全文搜索所有记忆文件
   - **低效点**：随着记忆增长，检索时间增加
   - **优化**：建立记忆索引文件，按主题分类存储

4. **学习成果转化效率**
   - **现状**：GitHub学习生成报告后需人工提取因子
   - **低效点**：学习→应用的链路长
   - **优化**：建立"学习-提取-验证-入库"自动化流水线

### 4.3.1 多策略组合优化

**当前问题**：单策略运行，缺乏组合视角
**优化目标**：构建风险分散的组合

**组合优化模型**：
```python
import numpy as np
from scipy.optimize import minimize

def portfolio_optimization(returns, cov_matrix, target_return=None):
    """
    均值-方差组合优化
    
    Args:
        returns: 各策略预期收益率向量
        cov_matrix: 收益率协方差矩阵
        target_return: 目标收益率（可选）
    """
    n = len(returns)
    
    # 目标函数：最小化组合方差
    def portfolio_variance(weights):
        return weights.T @ cov_matrix @ weights
    
    # 约束条件
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},  # 权重和为1
    ]
    
    if target_return:
        constraints.append(
            {'type': 'eq', 'fun': lambda w: w.T @ returns - target_return}
        )
    
    # 权重边界：0-100%
    bounds = [(0, 1) for _ in range(n)]
    
    # 优化
    result = minimize(portfolio_variance, 
                     x0=np.array([1/n]*n),
                     method='SLSQP',
                     bounds=bounds,
                     constraints=constraints)
    
    return result.x
```

### 4.3.2 风险平价配置

**风险平价逻辑**：各策略对组合风险贡献相等

```python
def risk_parity_weights(cov_matrix):
    """计算风险平价权重"""
    n = cov_matrix.shape[0]
    
    def risk_budget_objective(weights):
        """风险预算目标函数"""
        portfolio_vol = np.sqrt(weights.T @ cov_matrix @ weights)
        marginal_risk = (cov_matrix @ weights) / portfolio_vol
        risk_contrib = weights * marginal_risk
        target_risk = portfolio_vol / n
        return np.sum((risk_contrib - target_risk)**2)
    
    result = minimize(risk_budget_objective,
                     x0=np.array([1/n]*n),
                     method='SLSQP',
                     bounds=[(0.05, 0.95) for _ in range(n)],
                     constraints={'type': 'eq', 'fun': lambda w: np.sum(w) - 1})
    
    return result.x
```
