**设计原则**: 
- Heartbeat 负责 30分钟巡检（仅异常告警）
- Cron 负责精确时点任务（不重复 heartbeat 的工作）
- 所有任务 lightContext=true 节省 Token
- 正常从不推送，避免噪音
