## 🚀 快速开始

### 方法一：一键启动（推荐）

双击运行 `启动V4策略.bat`

这将自动：
1. 启动V4策略引擎
2. 启动监控面板 (http://localhost:8081)
3. 打开浏览器查看监控

### 方法二：手动启动

```bash
# 1. 启动监控面板
python v4_monitor.py

# 2. 在另一个窗口启动策略引擎
python src/v4_live_engine.py
```

## 📊 策略逻辑

### 三重信号机制

1. **V3涡轮信号**: 价格偏离20日均线5%
2. **均值回归信号**: Z-Score超过±2.0
3. **ML预测信号**: 机器学习趋势预测

### 双重确认规则

任意**两个**信号为真时执行交易：
- 买入: (V3 + ML) 或 (V3 + MR) 或 (ML + MR)
- 卖出: (V3 + ML) 或 (V3 + MR) 或 (ML + MR)
